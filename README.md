# nenv

> A scaffold project from infore.com

## Build Setup

``` bash
# install nenv
cnpm install git+http://192.168.37.5:nenv/nenv.git -g

# init project
nenv init [<dir>] -n name

# install dependencie
cnpm install

# serve with hot reload at localhost:8880
npm run dev

# build for production with minification
npm run build

```
